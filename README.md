# BoyiaUI引擎

## 本引擎为自主研发，主要借鉴部分W3C标准思想和CSS样式匹配思想
## 本引擎为自主原创
## 1，包含布局引擎，widget组件主要作用为组件布局和绘制
## 2，包含渲染引擎，painter由opengl进行独立线程绘制
## 3，包含脚本引擎，boyia脚本虚拟机运行boyia脚本来处理UI创建和业务逻辑
